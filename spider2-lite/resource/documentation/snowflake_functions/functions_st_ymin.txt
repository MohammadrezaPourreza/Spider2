  ST_YMIN  ¶

Returns the minimum latitude (Y coordinate) of all points contained in the
specified [ GEOGRAPHY  ](../data-types-geospatial.html#label-data-types-
geography) or [ GEOMETRY  ](../data-types-geospatial.html#label-data-types-
geometry) object.

##  Syntax  ¶

    
    
    ST_YMIN( <geography_or_geometry_expression> )
    

Copy

##  Arguments  ¶

` _ geography_or_geometry_expression  _ `

    

The argument must be an expression of type GEOGRAPHY or GEOMETRY.

##  Returns  ¶

Returns a REAL value.

##  Usage notes  ¶

  * The function takes into account the curvature of the edges toward the poles.
