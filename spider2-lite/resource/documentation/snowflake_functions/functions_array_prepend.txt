  ARRAY_PREPEND  ¶

Returns an array containing the new element as well as all elements from the
source array. The new element is positioned at the beginning of the array.

See also:

    

[ ARRAY_APPEND  ](array_append) , [ ARRAY_INSERT  ](array_insert)

##  Syntax  ¶

    
    
    ARRAY_PREPEND( <array> , <new_element> )
    

Copy

##  Arguments  ¶

` _ array  _ `

    

The source array.

` _ new_element  _ `

    

The element to be prepended.

##  Returns  ¶

This returns the updated array.

##  Usage notes  ¶

  * When you pass a [ structured ARRAY  ](../data-types-structured) to the function, the function returns a structured ARRAY of the same type. 

  * If ` _ array  _ ` is a [ structured ARRAY  ](../data-types-structured) , the type of the new element must be [ coercible  ](../data-types-structured.html#label-structured-types-casting-implicit) to the type of the ARRAY. 

##  Examples  ¶

The example below shows that the prepended element is placed at the beginning
of the array:

>
>     SELECT ARRAY_PREPEND(ARRAY_CONSTRUCT(0,1,2,3),'hello');
>     +-------------------------------------------------+
>     | ARRAY_PREPEND(ARRAY_CONSTRUCT(0,1,2,3),'HELLO') |
>     |-------------------------------------------------|
>     | [                                               |
>     |   "hello",                                      |
>     |   0,                                            |
>     |   1,                                            |
>     |   2,                                            |
>     |   3                                             |
>     | ]                                               |
>     +-------------------------------------------------+
>  
>
