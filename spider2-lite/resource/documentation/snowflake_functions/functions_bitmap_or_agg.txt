  BITMAP_OR_AGG  ¶

Returns a bitmap containing the results of a binary OR operation on the input
bitmaps.

See also:

    

[ Using Bitmaps to Compute Distinct Values for Hierarchical Aggregations
](../../user-guide/querying-bitmaps-for-distinct-counts)

##  Syntax  ¶

    
    
    BITMAP_OR_AGG( <bitmap> )
    

Copy

##  Arguments  ¶

` _ bitmap  _ `

    

A bitmap returned by the [ BITMAP_CONSTRUCT_AGG  ](bitmap_construct_agg) or
BITMAP_OR_AGG function.

##  Returns  ¶

The function returns a bitmap containing the results of a binary OR operation
on the input bitmaps.
