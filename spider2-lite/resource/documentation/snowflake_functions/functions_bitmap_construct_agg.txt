  BITMAP_CONSTRUCT_AGG  ¶

Returns a bitmap with bits set for each distinct value in a group.

See also:

    

[ Using Bitmaps to Compute Distinct Values for Hierarchical Aggregations
](../../user-guide/querying-bitmaps-for-distinct-counts)

##  Syntax  ¶

    
    
    BITMAP_CONSTRUCT_AGG( <relative_position> )
    

Copy

##  Arguments  ¶

` _ relative_position  _ `

    

The relative position of a bit for a value (returned by the [
BITMAP_BIT_POSITION  ](bitmap_bit_position) function).

##  Returns  ¶

The function returns a BINARY value that is a bitmap with bits set for each
distinct value in a group.
