  CURRENT_IP_ADDRESS  ¶

Returns the IP address of the client that submitted the request.

##  Syntax  ¶

    
    
    CURRENT_IP_ADDRESS()
    

Copy

##  Arguments  ¶

None.

##  Examples  ¶

Return the current IP address of the client that is connected to Snowflake:

>
>     select current_ip_address();
>  
>     +----------------------+
>     | CURRENT_IP_ADDRESS() |
>     +----------------------+
>     | 192.0.2.255          |
>     +----------------------+
>  
>
> Copy
