  ST_ISVALID  ¶

Returns TRUE if the specified GEOGRAPHY or GEOMETRY object represents a [
valid shape  ](../data-types-geospatial.html#label-geospatial-invalid-shape-
handling-intro) .

##  Syntax  ¶

    
    
    ST_ISVALID( <geography_or_geometry_expression> )
    

Copy

##  Arguments  ¶

` _ geography_or_geometry_expression  _ `

    

The argument must be an expression of type GEOGRAPHY or GEOMETRY.

##  Returns  ¶

Returns a BOOLEAN.
